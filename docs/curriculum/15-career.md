Turning skill into offers: portfolio, resume, interview formats, and the
working habits of real data teams.

## Learning outcomes

- Build a portfolio and resume that pass a hiring manager's 30-second scan
- Recognize every DS interview format and prepare deliberately for each
- Handle SQL, pandas, statistics/ML, and system-design interview rounds
- Operate professionally: code review, stakeholders, ethics, first 90 days
